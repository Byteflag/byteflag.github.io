Technical

Offline working, by design

A mobile-optimised copy of your master data is stored on the mobile device. All app configuration is stored locally too. Provides high performance and seamless working in bad signal areas.

Unrivalled synchronisation technology

Our app supports several different kinds of sync for various different types of data, all of which are some variant of a “incremental” and/or “differential” protocol. All sync data is compressed with a “LZMA” algorithm to ensure the absolute best bandwidth savings.

Native app experience

A true native app experience that offers the best performance, look & feel and able to take advantage of the latest hardware platform features.

Integration

Data sources

PanConnect can pull your master data from almost any source, including:

  • ODBC-compatible databases like MSSQL, SQLite, MySQL, PostgreSQL, Progress, Ingres, and hundreds more.
  • XML, JSON, CSV files
  • Excel spreadsheets & Access databases
  • REST and SOAP web services
  • E-mail
  • PDF “contiguous pages, print run” files with an intelligent pattern-matching based page splitting logic

Special integrations

We've created some bespoke integration modules for certain third-party systems and markets to save you time, including:

  • Social Housing databases: Capita, Civica, Northgate, Orchard and many more.
  • Electronic document stores: Capita Swordfish, Azure DocumentDB, Amazon Web Services and more.
  • Our MacroTron tool, a screen oriented macro engine, that can replicate the actions of a human; for those trickiest of systems that lack any integration points.

Encrypted, efficient, fault tolerant

Our integration toolset was designed from the outset to be highly secure, efficient and reliable.

All communications between your on-premise servers and our servers (if you host with us) are conducted across a highly reliable messaging service bus that is 2048-bit SSL secured.

Everything is compressed using LZMA algorithm to save massive amounts of bandwidth.
